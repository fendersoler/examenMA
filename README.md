Este proyecto contiene una calculadora de numeros naturales con 4 funcionalidades:
- Suma 
- Multiplicación 
- Resta 
- No resta
- Números romanos

Los sprints en los que se divide la calculadora son:
Sprint 1: Suma
Sprint 2: Multiplicación 
Sprint 3: Resta y No resta: decidimos con cliente unir ambas historias de usuario y crear una sola
Sprint 4: Números romanos
Sprint 5: pruebas de integración de todas las historias de usuario
